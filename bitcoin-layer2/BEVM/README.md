# BEVM

![BEVM logo](./BEVM.png)

BEVM is an EVM-compatible sidechain built on substrate. On the BEVM chain, the BTC is held in custody by a federated signer set.

Attributes:
- EVM-compatible
- BTC-backed
- Mainnet
- Sidechain
- WBTC Fee Token
